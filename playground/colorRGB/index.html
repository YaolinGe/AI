RGB Color Theory

RGB is an additive color model where red, green, and blue light are added together to create various colors. Each component ranges from 0 to 255.

  • When all components are 0, the result is black
  • When all components are 255, the result is white
  • Equal values of all components create shades of gray
  • Primary colors: Red (255,0,0), Green (0,255,0), Blue (0,0,255)
  • Secondary colors: Yellow (255,255,0), Cyan (0,255,255), Magenta (255,0,255)
